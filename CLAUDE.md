# Dart Desk

Flutter widget library for building CMS studio interfaces.

## Tech Stack

- **shadcn_ui** for UI components
- **signals** for reactive state management
- **zenrouter** for navigation
- **marionette** for UI testing/automation

## Running the CMS App

Use the dart MCP tools (`mcp__dart__launch_app`, `mcp__dart__list_devices`) to launch the app — do not use manual `flutter run` commands.

## Agent Model Selection

- **Testing/verification:** Use sonnet subagents (fast, cheap)
- **Debugging/writing code:** Use opus (capable, thorough)
